Task Descriptor: Criar mapa de tarefas
Alinhar na melhor sequência as atividades que criam valor (Fluxo de Valor)
Based on Method Task: Criar mapa de tarefas
Main Description

Fluxos de valor são as atividades necesárias para a produção de todos os produtos.

Identifique o fluxo de valor, o conjunto de  todas as ações necessárias para finalizar o produto, incluindo todo o fluxo de produção, desde as primeiras validações do problema até a entrega do produto final. Depois de criar o mapa identifique os possíveis desperdícios de tempo e os elimine.
